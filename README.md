## 👊 슈퍼 꺅두기 (SuperKkyakdugi)

🏏 V26.05.07

● 본 게임은 **2D 플랫포머 게임**으로, 툴라 작가님의 '꺅두기'의 리소스를 사용해 제작한 꺅두기 팬 게임입니다.
